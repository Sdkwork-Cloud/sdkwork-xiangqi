# SDKWork Xiangqi SDK Families

Generated SDK families for the XIANGQI application:

| Family | Authority | Generated client |
| --- | --- | --- |
| `sdkwork-xiangqi-app-sdk` | `sdkwork-xiangqi-app-api` | `@sdkwork-internal/xiangqi-app-sdk-generated` |
| `sdkwork-xiangqi-backend-sdk` | `sdkwork-xiangqi-backend-api` | `@sdkwork-internal/xiangqi-backend-sdk-generated` |

## Commands

```bash
pnpm api:materialize
pnpm sdk:generate
pnpm sdk:build
pnpm sdk:generate:check
```

Route manifests live under `sdks/_route-manifests/` and are owned by `sdkwork-routes-*` crates.
